# Google-Classroom-AutoComment
Automate the commenting on any post in a Google Classroom with Katalon Studio Browser Extension
<hr>
<h3>Steps:</h3>
<ul>
  <li>Install Katalon Studio Browser Extension on your browser
  <li>Download this script
  <li>Change the link to your own Google Classroom
  <li>Update other placeholders in the script
  <li>Run the script to automate the process!!!
</ul>
